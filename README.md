## The PM Analytics Principle

Anyone can run a query. A PM asks: **"What should the team do differently because of this data?"**

Every project in this repository follows the same structure:
1. Business question (what decision does this inform?)
2. Dataset and methodology
3. Analysis
4. Key findings
5. **Product recommendation** — *therefore the team should...*
6. Lessons learned
